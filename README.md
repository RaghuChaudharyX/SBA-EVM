# SBA-EVM
Secured Biometric Aadhar-Linked Electronic Voting Machine

Team Name: Wild Smashers
Team No: 213

📌 Project Overview

SBA-EVM (Secure Biometric Aadhaar-Linked Electronic Verification Model) is a secure digital voter verification system designed to improve the transparency and security of electronic voting. The system verifies voter identity using biometric authentication linked with Aadhaar, ensuring that only eligible voters can participate in the voting process.

The system prioritizes privacy and security by performing authentication through the UIDAI API without storing sensitive Aadhaar or biometric data locally.

🎯 Problem Statement

Traditional voting systems often face challenges such as:

Voter impersonation

Identity fraud

Data privacy concerns

Inefficient verification processes

A secure and reliable method is required to verify voter identity while protecting personal information.

💡 Our Solution

SBA-EVM provides a secure authentication-based verification system that confirms voter identity using Aadhaar-linked biometric authentication.

Key aspects of our solution:

Aadhaar-based authentication using UIDAI API

No storage of voter Aadhaar or biometric data locally

Encrypted authentication process

Secure and fast voter verification

Privacy-focused identity validation

⚙️ How It Works

Voter enters their Aadhaar details for verification.

The system sends an authentication request through the UIDAI API.

Biometric verification is performed securely.

UIDAI confirms whether the identity is valid.

If verified, the voter is allowed to proceed with voting.

The system only performs authentication and does not store sensitive data.

🔐 Security & Privacy Features

No local storage of Aadhaar or biometric data

Secure encrypted authentication process

Tamper-resistant verification system

Privacy-focused identity validation
